## 1. 📱 Що таке Telegram? <a name="що-таке-telegram"></a>  

### 1.1 Загальна інформація  
- 🧠 Короткий огляд: месенджер з акцентом на приватність, швидкість і зручність  
- ⚖️ Відмінності від інших платформ (WhatsApp, Viber, Discord)  
- 🧩 Формати контенту: канали, чати, боти, сторіс, реакції  

### 1.2 Створення Telegram-каналу та чату  
- 🛡️ Типи каналів: публічний / приватний  
- 🏗️ Як створити канал:
  - 📝 Назва
  - 🆔 @юзернейм (якщо публічний)
  - 🖼️ Аватар, опис  
- 💬 Як створити пов'язаний чат:
  - 🔗 Прив’язка чату до каналу
  - ⚙️ Налаштування прав: хто може писати, хто модерувати
  - 📣 Сповіщення: автоматичне публікування постів у чат  

### 1.3 Налаштування опису, біо та зв’язки  
- ✍️ Опис каналу — як зробити його привабливим  
- 👤 Біо адміністраторів / авторів  
- 🔄 Зв’язок каналу та чату  
- 🎨 Поради зі стилю оформлення: емодзі, посилання, заклики до дії  

### 1.4 Визначення концепції  
- 🗂️ Тематика каналу (новини, блог, меми, підбірки)  
- 🔍 Як обрати нішу  
- 📚 Приклади форматів  
- 🗣️ Тон спілкування

## 6. 🧾 Форматування та оформлення контенту <a name="#форматування-та-оформлення-контенту"></a>  

Гарно оформлений пост — це не лише про естетику, а й про ефективність: такі пости краще читають, зберігають і пересилають.

### 6.1 🔤 Форматування тексту

- **Жирний текст** — виділяй ключове  
- _Курсив_ — для акцентів або пояснень  
- `Моноширинний` — для коду або виділень  
- 💬 Цитати — зручно для думок, зворотного зв’язку  
- 📌 Використовуй заголовки, абзаци, відступи для читабельності  
- #️⃣ Хештеги — 1–3 ключових, бажано в кінці  
- 🔗 Посилання:
  - Telegram: `t.me/username`
  - Зовнішні: скорочуй через bit.ly або вставляй у текст

### 6.2 🖼️ Оформлення зображень

- 📐 Пропорції:
  - 1:1 — універсальні (постер, іконка)
  - 16:9 — горизонтальні (панорама, обкладинка)
  - 9:16 — вертикальні (сторіс, інфографіка)
- 🖼️ Формати файлів:
  - JPEG — для фото, фонів
  - PNG — для прозорих зображень, схем, графіки
- 🎨 Поради:
  - Завжди вирівнюй по центру (якщо текст поверх)
  - Уникай перенасичення дрібними елементами
  - Залиш візуальний “повітря” — не перевантажуй

### 6.3 🧩 Шаблони та приклади

- 📌 Структура допису:
  🔔 Заголовок  
  Короткий вступ — про що пост  
  🔹 Основна частина — по пунктах або абзацами  
  ✅ Висновок або заклик до дії  
  #хештег #тема

- 📸 Типові шаблони для візуалу:
  - Колаж 2–4 картки
  - Картинка + заголовок текстом зверху
  - Прев’ю до відео
  - Мінімалістичний мем або комікс-формат
